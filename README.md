# 100kinSAT
100kinSAT - CanSat（カンサット）競技の入門キット

詳細は[Wiki](https://github.com/ymt117/100kinSAT/wiki)を参照

### 概要
---

CanSatは「Can（缶）」と「Satellite（サテライト，衛星）」を組み合わせた造語で,
CanSatと呼ばれる小型模擬人工衛星による惑星探査を模した競技です．

CanSatの製作を通して，衛星モデルの開発を学ぶことができます．
